authManagerForEnvironment:study:baseURLPath:

+ (instancetype)authManagerForEnvironment:(SBBEnvironment)environment 
        study:(NSString *)study baseURLPath:(NSString *)baseURLPath;
Parameters
environment

The SBBEnvironment to use (prod, staging, dev).

study

The app-specific study identifier to use (typically set in gSBBAppStudy).

baseURLPath

The URL path to prefix with the app server prefix and environment string (e.g. @"sagebridge.org")

Return Value

An SBBAuthManager component configured for an environment, appURLPrefix, and baseURLPath.

Discussion

Return an SBBAuthManager component configured for the specified environment, appURLPrefix, and baseURLPath with a default network manager.

Use this method directly only if you need to redirect your Bridge API accesses to a test server.

authManagerWithBaseURL:

+ (instancetype)authManagerWithBaseURL:(NSString *)baseURL;
Parameters
baseURL

The baseURL to use.

Return Value

An SBBAuthManager component configured with a specific network manager.

Discussion

Return an SBBAuthManager component configured for the specified baseURL with a default network manager.

Use this if you need to test against a custom server. Implies a custom environment for credential storage purposes.

authManagerWithNetworkManager:

+ (instancetype)authManagerWithNetworkManager:(id<SBBNetworkManagerProtocol>)networkManager;
Parameters
networkManager

The SBBNetworkManager to use.

Return Value

An SBBAuthManager component configured with a specific network manager.

Discussion

Return an SBBAuthManager component configured with the specified network manager.

Use this if you need to run with a custom network manager. Also implies a custom environment for credential storage purposes.

defaultComponent

+ (instancetype)defaultComponent;
Return Value

The default (shared) SBBAuthManager component.

Discussion

Return the default (shared) component of this type (SBBAuthManager), configured with the gSBBAppURLPrefix and the default environment. In debug builds, this is SBBEnvironmentStaging; in release builds, SBBEnvironmentProd. Also configures the component to use the SBBNetworkManager currently registered the first time this is called, or the default if none was registered yet.
