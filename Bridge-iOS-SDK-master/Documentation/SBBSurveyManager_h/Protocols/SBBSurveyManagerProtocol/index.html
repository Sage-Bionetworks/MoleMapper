Introduction

This protocol defines the interface to the SBBSurveyManager's non-constructor, non-initializer methods. The interface is abstracted out for use in mock objects for testing, and to allow selecting among multiple implementations at runtime.

submitAnswers:toSurvey:completion:

- (NSURLSessionDataTask *)submitAnswers:(NSArray *)surveyAnswers 
        toSurvey:(SBBSurvey *)survey completion:(SBBSurveyManagerSubmitAnswersCompletionBlock)completion;
Parameters
surveyAnswers

An NSArray of survey answer objects for the questions answered.

survey

The SBBSurveyObject representing the survey being answered, obtained e.g. from one of the getSurveyBy... methods of this manager.

completion

An SBBSurveyManagerSubmitAnswersCompletionBlock to be called upon completion. The identifierHolder passed in contains an identifier assigned by the server to the survey response created by submitting these answers.

Return Value

An NSURLSessionDataTask object so you can cancel or suspend/resume the request.

Discussion

Submit a set of answers to a survey specified by an SBBSurvey object.

submitAnswers:toSurvey:withResponseIdentifier:completion:

- (NSURLSessionDataTask *)submitAnswers:(NSArray *)surveyAnswers 
        toSurvey:(SBBSurvey *)survey withResponseIdentifier:(NSString *)identifier 
        completion:(SBBSurveyManagerSubmitAnswersCompletionBlock)completion;
Parameters
surveyAnswers

An NSArray of survey answer objects for the questions answered.

survey

The SBBSurveyObject representing the survey being answered, obtained e.g. from one of the getSurveyBy... methods of this manager.

identifier

An identifier to use for the SurveyResponse created as a result of submitting these answers. If nil, this behaves the same as calling submitAnswers:toSurveyByRef:completion:.

completion

An SBBSurveyManagerSubmitAnswersCompletionBlock to be called upon completion. The identifierHolder passed in contains the identifier of the survey response created by submitting these answers, which will match what you sent in the identifier parameter (unless you sent nil).

Return Value

An NSURLSessionDataTask object so you can cancel or suspend/resume the request.

Discussion

Submit a set of answers to a survey specified by an SBBSurvey object, and specify an identifier for the SurveyResponse object to create as a result.
