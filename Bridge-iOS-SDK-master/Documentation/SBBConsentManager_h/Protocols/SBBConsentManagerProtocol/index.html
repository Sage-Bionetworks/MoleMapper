Introduction

This protocol defines the interface to the SBBConsentManager's non-constructor, non-initializer methods. The interface is abstracted out for use in mock objects for testing, and to allow selecting among multiple implementations at runtime.

consentSignature:birthdate:signatureImage:dataSharing:completion:

- (NSURLSessionDataTask *)consentSignature:(NSString *)name 
        birthdate:(NSDate *)date signatureImage:(UIImage*)signatureImage 
        dataSharing:(SBBConsentShareScope)scope completion:(SBBConsentManagerCompletionBlock)completion;
Parameters
name

The user's name.

date

The user's birthday in the format "YYYY-MM-DD".

signatureImage

Image file of the user's signature. Should be less than 10kb. Optional, can be nil.

scope

The scope of data sharing to which the user has consented.

completion

An SBBConsentManagerCompletionBlock to be called upon completion.

Return Value

An NSURLSessionDataTask object so you can cancel or suspend/resume the request.

Discussion

Submit the user's "signature" and birthdate to indicate consent to participate in this research project.

dataSharing:completion:

- (NSURLSessionDataTask *)dataSharing:(SBBConsentShareScope)scope 
        completion:(SBBConsentManagerCompletionBlock)completion;
Parameters
scope

The scope of data sharing to set for this user.

completion

An SBBConsentManagerCompletionBlock to be called upon completion.

Return Value

An NSURLSessionDataTask object so you can cancel or suspend/resume the request.

Discussion

Change the scope of data sharing for this user. This should only be done in response to an explicit choice on the part of the user to change the sharing scope.

resumeConsentWithCompletion:

- (NSURLSessionDataTask *)resumeConsentWithCompletion:(SBBConsentManagerCompletionBlock)completion __deprecated;
Parameters
completion

An SBBConsentManagerCompletionBlock to be called upon completion.

Return Value

An NSURLSessionDataTask object so you can cancel or suspend/resume the request.

Discussion

DEPRECATED: Resume the user's previously-suspended consent to participate.

WARNING:

This method is deprecated as of BridgeSDK version 2. You should call the dataSharing: method instead. This method is equivalent to calling dataSharing: with a scope of SBBConsentShareScopeStudy.

retrieveConsentSignatureWithCompletion:

- (NSURLSessionDataTask*)
        retrieveConsentSignatureWithCompletion:(SBBConsentManagerRetrieveCompletionBlock)completion;
Parameters
completion

An SBBConsentManagerRetrieveCompletionBlock to be called upon completion.

Return Value

An NSURLSessionDataTask object so you can cancel or suspend/resume the request.

Discussion

Retrieve the user's consent signature as previously submitted. If the user has not submitted a consent signature, this method throws an Entity Not Found error.
